# 15.1 Capstone Design & Scoping

Defines how to select, scope, and de-risk an AI capstone with clear outcomes, statement of work, and milestone planning.

## Start Here

1. Theory: `theory/15-1-capstone-design-and-scoping.md`
2. Notebook: `notebooks/15-1-capstone-design-and-scoping-planner.ipynb`

## Outcomes

- Build a practical project charter and statement of work.
- Score ideas using feasibility, impact, and career alignment criteria.
- Set realistic capstone success criteria and boundaries.

## Includes

- project selection and scoping templates,
- architecture pre-design template,
- milestone planning and definition-of-done patterns,
- capstone scoring rubric for early-stage quality checks.
