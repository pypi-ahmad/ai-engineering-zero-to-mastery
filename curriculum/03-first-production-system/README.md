# 03: First Shipped System (Your First End-to-End Loop)

Goal: go from “I trained a model” to “I shipped a small system with artifacts, metrics, and an API contract”.

This module is intentionally early. It prevents a common failure mode: spending months on theory and never shipping anything.

## Completion Criteria (Self-Check)

- You can generate artifacts locally (model file + metrics JSON).
- You can explain how to reproduce a run (metadata, versions, splits).
- You have one “regression gate” you would use to block a bad release.
