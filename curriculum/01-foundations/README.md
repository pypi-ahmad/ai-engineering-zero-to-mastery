## Why This Matters

Most “AI learning” fails because learners get stuck on:
- environments,
- debugging,
- data loading,
- reproducibility,
- and basic software habits.

This module prevents that.

## Completion Criteria (Self-Check)

- You can run a notebook from a clean kernel.
- You can explain `train/val/test` vs “train and hope”.
- You can write and run a small Python script in this repo (not just notebooks).
