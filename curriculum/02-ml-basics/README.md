# 02: ML Basics (Baselines, Metrics, and Evaluation Discipline)

Goal: learn the minimum ML you need to build systems responsibly: baselines, data splits, and evaluation.

## Prerequisites

- [01-foundations](../01-foundations/README.md)

## Do These Lessons

Core:
- Lesson 3 (required):
  - [`03-classical-machine-learning/README.md`](../../03-classical-machine-learning/README.md)

Support (recommended if you are new to math):
- Lesson 2 (selective, not perfection):
  - [`02-mathematics-for-ai/README.md`](../../02-mathematics-for-ai/README.md)

## Suggested Sub-Lesson Order (Beginner-First)

1. 3.1 Supervised learning (train a baseline)
2. 3.3 Evaluation and selection (metrics, CV, leakage)
3. 3.2 Unsupervised learning (optional early, useful later)

## Completion Criteria (Self-Check)

- You can explain why accuracy is not enough for many problems.
- You can identify leakage and prevent it with splits.
- You can produce a baseline model + metric report that you would be comfortable shipping as v0.
