# Maintainer Research (Not Committed)

This folder is intentionally kept **out of version control** (see `.gitignore`).

Reason:
- scraped pages and raw research dumps are noisy for beginners,
- they can contain machine-specific paths and large HTML/markdown payloads,
- they can create redistribution/copyright risk.

If you want to trace sources used to build the curriculum, use the curated registry:
- `docs/sources-and-references.md`

If you do maintainer research locally, keep raw dumps here and only promote **source URLs**
and **small summaries** into the references registry or lesson docs.
