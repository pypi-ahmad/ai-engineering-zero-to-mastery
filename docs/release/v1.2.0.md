# Release v1.2.0 (2026-07-03)

This release focuses on **portable, beginner-first learning** and **clean release hygiene**.

## Highlights

- No committed notebook-generated `data/` or `artifacts/` directories.
  - Learners generate artifacts locally; the repo stays lean and professional.
- Removed machine-specific absolute paths from notebooks/handbook excerpts.
- Added an offline fallback in Lesson 5.1 when MNIST download is unavailable.
- Added explicit exercises + solutions for Lessons 1–3 to make practice consistent.
- Tightened curriculum validation:
  - markdown links checked repo-wide,
  - generated/maintainer-only content forbidden from being tracked,
  - absolute repo paths rejected.

## How To Verify

```bash
uv venv --python 3.12
source .venv/bin/activate
uv sync --frozen --group dev
python3 scripts/validate_curriculum.py
uv run pytest -q
```

## Notes

- Notebooks are still validated structurally/syntactically in CI; full execution remains out of CI scope due to GPU/network variability.
