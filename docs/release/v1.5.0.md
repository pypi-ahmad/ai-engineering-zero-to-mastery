# Release v1.5.0 (2026-07-03)

This release focuses on tutorial-grade pedagogy: clearer definitions, stronger verification steps, and fewer beginner dead-ends.

## Highlights

- Added a beginner-friendly glossary: `docs/glossary.md`
- Strengthened onboarding and setup troubleshooting with explicit verification steps
- Upgraded Lesson 1–6 READMEs (and sub-lessons) with:
  - why this matters
  - key terms (plain English)
  - expected outcomes
  - verification steps
  - common mistakes
- Upgraded Lessons 7–15 lesson READMEs with expected outcomes + verification guidance
- Added `docs/notebook-playbook.md` and improved intros for selected beginner-critical notebooks

## How To Verify

```bash
uv venv --python 3.12
source .venv/bin/activate
uv sync --frozen --group dev
python3 scripts/validate_curriculum.py
uv run pytest -q
```
