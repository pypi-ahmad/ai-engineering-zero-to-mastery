# Release v1.4.0 (2026-07-03)

This release makes the repo easier for complete beginners to navigate while preserving the full depth of the 15-lesson curriculum.

## Highlights

- Added a guided, beginner-first learning path in `curriculum/`:
  - recommended module ordering,
  - prerequisite flow,
  - completion criteria (self-checks),
  - what to focus on vs what to postpone.
- Updated tracks and onboarding docs to route beginners to the guided path.
- Added a `projects/` index and made capstone quickstart commands reproducible with `uv run python`.

## How To Verify

```bash
uv venv --python 3.12
source .venv/bin/activate
uv sync --frozen --group dev
python3 scripts/validate_curriculum.py
uv run pytest -q
```
