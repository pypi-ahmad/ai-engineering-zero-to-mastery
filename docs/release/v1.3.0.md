# Release v1.3.0 (2026-07-03)

This release strengthens the **beginner practice loop** through Lesson 6 so learners get consistent, checkable practice alongside theory + notebooks.

## Highlights

- Added **exercises + solutions for Lessons 4–6**, covering:
  - Deep learning fundamentals (Lesson 4)
  - Generative models and LLMs (Lesson 5)
  - MLOps/LLMOps foundations for production AI systems (Lesson 6)
- Added a Lesson 6.5 **LLMOps practice set** focused on:
  - prompts as versioned artifacts,
  - evaluation harnesses and regression checks,
  - token/cost budget gates,
  - retrieval circuit breakers for RAG reliability,
  - tool allowlists and risk-gated execution.
- Updated curriculum validation so Lessons 1–6 must include:
  - `## Practice (Recommended)` section in each sub-lesson README
  - `exercises/exercises.md` and `exercises/solutions.md`

## How To Verify

```bash
uv venv --python 3.12
source .venv/bin/activate
uv sync --frozen --group dev
python3 scripts/validate_curriculum.py
uv run pytest -q
```

## Notes

- Notebooks are validated structurally/syntactically in CI; full execution remains out of CI scope due to GPU/network variability.
