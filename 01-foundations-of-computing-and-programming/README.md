# Lesson 1: Foundations of Computing and Programming

This module establishes the engineering baseline for the rest of the curriculum: writing reliable Python, choosing the right data structures, and shipping maintainable code.

## Learning Goals
- Build core Python fluency for AI and data workflows.
- Reason about data structures, algorithms, and complexity trade-offs.
- Apply software engineering practices (Git, testing, modular design, APIs).

## How It Fits in the Curriculum
Lesson 1 is the prerequisite for Lessons 2-15. It turns beginner coding skills into production-oriented engineering habits used in ML, GenAI, and agent systems.

## Sub-lessons
- `1.1 Programming Basics`
- `1.2 Data Structures & Algorithms`
- `1.3 Software Engineering Practices`

## Artifacts
Each sub-lesson includes:
- `theory/` chapter markdown (+ exported PDF)
- `notebooks/` runnable teaching notebook(s)
- local `README.md` with quick navigation

## How to Use This Lesson
1. Complete sub-lessons in order: `1.1 -> 1.2 -> 1.3`.
2. Read each `theory/` chapter first, then run `notebooks/`.
3. Reuse checklists/templates in your own scripts and mini-projects.

## Bridge to Lesson 2
**Why this follows:** after coding foundations, you need mathematical intuition to reason about model behavior and optimization.

**You should already know:** Python basics, data structures, and software engineering workflows.

**What you will do next:** learn linear algebra, calculus, probability, and applied statistics for ML decisions.
