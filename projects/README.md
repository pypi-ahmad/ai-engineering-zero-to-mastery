# Projects

This folder contains runnable project scaffolds that turn lessons into portfolio-grade artifacts.

## Recommended Order (Beginner-First)

1. Start with the capstone scaffold:
   - [`capstone-template/README.md`](./capstone-template/README.md)

## Why This Exists

Many learners can complete notebooks but still struggle to:
- structure code as scripts/modules,
- save artifacts and run metadata,
- define an API contract,
- add tests and verification.

Projects are where those skills become real.
